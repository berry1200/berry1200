## 🚀 Key Projects

### 🤖 Autonomous Mobile Robot — TurtleBot3 Smart Office
> `ROS 2` · `Gazebo` · `SLAM` · `A* Path Planning` · `DWA Planner` · `Computer Vision` · `TurtleBot3 Waffle Pi`

Designed and implemented a **complete autonomous navigation system** in a simulated smart office environment — integrating SLAM for real-time mapping, A* global path planning, and DWA local obstacle avoidance. Added a **computer vision module** for human/object detection to enable context-aware robot behaviour in dynamic spaces. Validated full navigation stack performance across multiple test scenarios.

---

### 🧠 Human Detection for Mobile Robots — CNN Transfer Learning
> `MATLAB` · `ResNet-18` · `Transfer Learning` · `Data Augmentation` · `Precision/Recall/F1`

Fine-tuned **ResNet-18** for binary human/non-human image classification, achieving **94.17% test accuracy**. Built and augmented a custom dataset (including low-light transformations) for improved robustness on resource-constrained robot platforms. Evaluated with full confusion matrix and classification report analysis.

---

### 🦾 NAO Robot Physiotherapy Assistant *(Group Project)*
> `NAO Robot` · `Choreography Software` · `Motion Planning` · `Human-Robot Interaction`

Led the **motion design and choreography** component — programming physiotherapy-guided movement sequences with smooth, safe transitions to mimic clinically informed exercises. Integrated motion control, speech, and sensor feedback into a coherent assistive robot behaviour with a focus on user safety and interaction quality.

---

### 🛡️ Noise-Resilient NLP Safety Layer for Robot Interaction *(MSc Dissertation)*
> `Python` · `NLP` · `Confidence Scoring` · `Robot Safety` · `Hallucination Prevention`

Developing a **safety middleware layer** that evaluates NLP model outputs using confidence scoring to prevent robots from executing unrecognised or hallucinated commands. Designing thresholding and fallback logic to ensure robots default to **safe states** under uncertainty — contributing to responsible AI deployment in human-robot interaction contexts.
